This repository contains all data and scripts to reproduce the results of:
Julian Echave, Eleisha L. Jackson, Claus O. Wilke. Relationship between protein thermodynamic constraints and variation of evolutionary rates among sites. Phys. Biol. 12:025002, 2015.

Contents:

Huang_et_al_data/
	Contains the data from Huang et al. (2014) 
ddG_calculations/
	Contains the scripts, data, files etc associated with the rosetta and FoldX ddG calculations
empirical_rate_calculations/
	Contains the scripts, data, files etc associated with the calculation of the evolutionary rates using Rate4Site
model_rate_calculations/
        Contains the scripts and output associated with the calculation of the evolutionary rates predicted by the stability model and the stress model
