This is a directory that contains all of the files needed to perform the evolutionary rate calculation analysis on the dataset. 

Contents:

	evolutionary rates using Rate4Site. See the directories README.txt file for a list of contents.
	
protein alignments/
	This is a directory containing the protein alignments used for the analysis. See the directory README.txt
